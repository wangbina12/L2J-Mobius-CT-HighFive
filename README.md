# L2J-Mobius-CT-HighFive

这是一个 L2J Mobius Classic 2.6 High Five 服务器项目。

## 项目介绍

本项目是基于 L2J Mobius 的 Lineage 2 Classic 2.6 High Five 私服项目。

## 功能特性

- 基于 L2J Mobius 核心
- Classic 2.6 High Five 版本
- 完整的游戏功能
- 自定义内容和功能

## 安装说明

1. 克隆仓库
```bash
git clone https://github.com/wangbina12/L2J-Mobius-CT-HighFive.git
```

2. 配置数据库

3. 运行服务器

## 项目结构

- `java/` - Java 源代码
- `dist/` - 分发文件
- `SQL/` - 数据库脚本
